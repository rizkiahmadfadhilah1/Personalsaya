Bagaimana langkah-langkah awal untuk memulai belajar membuat website ?

HTML

HTML adalah singkatan dari "Hypertext Markup Language" (Bahasa Markup Hiperteks). Ini adalah bahasa markup standar yang digunakan untuk membuat dan merancang halaman web.Dalam HTML, sebuah dokumen dibuat dengan menandai elemen-elemen tertentu menggunakan tag-tag.
learn more

CSS

CSS adalah singkatan dari "Cascading Style Sheets" (Lembar Gaya Cascading). Ini adalah bahasa gaya yang digunakan untuk mengendalikan tata letak (layout), tampilan, dan gaya suatu dokumen HTML. CSS memungkinkan pengembang web untuk memisahkan struktur dan konten dari presentasi visual, sehingga memungkinkan pengendalian yang lebih fleksibel dan konsisten terhadap tampilan halaman web.
learn more

Javascript

JavaScript adalah bahasa pemrograman tingkat tinggi yang umumnya digunakan untuk membuat dan mengendalikan perilaku dinamis pada halaman web. Berbeda dengan HTML dan CSS yang digunakan untuk membangun struktur dan tampilan halaman, JavaScript memberikan kemampuan interaktivitas dan dinamika pada website.
